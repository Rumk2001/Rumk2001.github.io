The Porject is an Asp.net project following the MVC principles. The website was created for imaginary pizza comapny looking to get a new website to support their business. The website it self is designed to allow customers to be able to add items to a shopping cart as a guest or as signed in individual. This page allows them be able to place orders for customers, allow employees to place orders for customers, cooks and delivery to see orders and be able to place them or change the status.

This website was made by Kevin Rumboldt. A student from the John abbott College Computer Science program.
